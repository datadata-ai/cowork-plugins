# DATADATA Cowork Plugins

Cowork plugins by [DATADATA](https://data-data.ai) — B2B intelligence, research, and data workflows.

## Install

In Claude Cowork: **Customize → Add marketplace** → paste:
```
datadata-ai/cowork-plugins
```

## Plugins

### Explorium Connector
B2B intelligence powered by [Explorium](https://www.explorium.ai/). Research companies, enrich prospects, and surface business events.

**Skills included:**
- `explorium-research` — Two-phase company research: structured enrichment + Web-Search-Events (AI-powered event discovery from web sources)
- `expansion-upsell-intelligence` — Account-specific expansion signal detector. Finds growth signals at a target company and generates a tailored upsell brief based on your product context.

**Requirements:** Explorium API key ([get one here](https://www.explorium.ai/))
